AetherVault - Live Project Links

AetherVault is a full-stack secure document management system built for the Database Systems Engineering and Distributed Backend Development course. It includes JWT authentication, role-based access control, document upload and download, category management, PostgreSQL storage, MongoDB-based semantic search, and a FastAPI gateway connecting the frontend with the backend services.
