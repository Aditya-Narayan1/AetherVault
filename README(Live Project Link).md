Short Description:

AetherVault is a full-stack secure document management system built for the Database Systems Engineering and Distributed Backend Development course. It supports user authentication, role-based access control, document upload/download, category management, PostgreSQL metadata storage, MongoDB semantic search storage, and a FastAPI gateway connecting the frontend with the backend services.
